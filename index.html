Quick Start Guide

1Clone the Repository

git clone https://github.com/PQC-Blockchain/quantum-wallet.git cd quantum-wallet

2Install Dependencies

pip install -r requirements.txt

3Configure Your Node

# Create .env file echo "NODE_ADDRESS=0.0.0.0" > .env echo "NODE_PORT=5000" >> .env echo "MINING_ENABLED=true" >> .env

4Start Mining

python pqc_blockchain_server_enhanced.py
